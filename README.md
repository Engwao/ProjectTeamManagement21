# 🚀 TEAM.CONTROL

**TEAM.CONTROL** — веб-приложение для управления проектными командами, разработанное в рамках MVP. Система позволяет регистрировать команды, вести паспорт проекта, управлять задачами и отслеживать прогресс выполнения проекта.

---

## 📌 Возможности

### Регистрация проектных команд
- создание новой команды;
- просмотр списка всех проектов;
- выбор активного проекта.

### Паспорт проекта
Хранение всей информации о проекте:

- название проекта;
- направление;
- сроки реализации;
- заказчик / куратор;
- руководитель проекта;
- команда проекта;
- цель по SMART;
- задачи проекта;
- используемые инструменты;
- бюджет;
- ограничения;
- риски;
- ожидаемый продукт;
- KPI.

### Управление задачами
- создание задач;
- изменение статуса выполнения;
- удаление задач;
- установка срока выполнения.

### Отслеживание прогресса
- автоматический расчет процента выполнения проекта;
- количество выполненных задач;
- общее количество задач;
- визуальный индикатор прогресса.

### Управление участниками
- добавление участников;
- удаление участников;
- хранение контактной информации и ролей.

### Управление этапами
- создание этапов проекта;
- изменение статуса этапов;
- удаление этапов.

# ⚙ Установка

## 1. Клонирование репозитория

```bash
git clone https://github.com/Engwao/ProjectTeamManagement21.git
```

или

```bash
git clone git@github.com:Engwao/ProjectTeamManagement21.git
```

---

## 2. Переход в папку проекта

```bash
cd ProjectTeamManagement21
```

---

## 3. Создание виртуального окружения

Windows

```bash
python -m venv venv
```

Linux / macOS

```bash
python3 -m venv venv
```

---

## 4. Активация окружения

Windows

```bash
venv\Scripts\activate
```

Linux / macOS

```bash
source venv/bin/activate
```

---

## 5. Установка зависимостей

```bash
pip install fastapi uvicorn python-multipart pydantic
```

---

## 6. Запуск приложения

Из корневой папки проекта:

```bash
python -m uvicorn backend.app.main:app --reload
```


После запуска приложение будет доступно по адресу:

```
http://127.0.0.1:8000
```

# 📡 REST API

## Проекты

| Метод | URL | Описание |
|--------|-----|----------|
| GET | `/groups` | Получить список проектов |
| POST | `/groups` | Создать проект |
| GET | `/groups/{id}` | Получить проект |
| PATCH | `/groups/{id}` | Обновить проект |
| DELETE | `/groups/{id}` | Удалить проект |

---

## Участники

| Метод | URL | Описание |
|--------|-----|----------|
| POST | `/users` | Добавить участника в группу |
| DELETE | `/users/{id}` | Удалить участника |

---

## Задачи

| Метод | URL | Описание |
|--------|-----|----------|
| POST | `/tasks` | Создать задачу |
| PATCH | `/tasks/{id}` | Обновить статус задачи |
| DELETE | `/tasks/{id}` | Удалить задачу |

---

## Этапы

| Метод | URL | Описание |
|--------|-----|----------|
| POST | `/stages` | Создать этап |
| PATCH | `/stages/{id}` | Обновить статус этапа |
| DELETE | `/stages/{id}` | Удалить этап |

---

## Прогресс

| Метод | URL | Описание |
|--------|-----|----------|
| GET | `/groups/{id}/progress` | Получить прогресс группы |
